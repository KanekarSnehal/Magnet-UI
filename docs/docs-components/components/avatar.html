Avatar

Avatars are images that users can set as their profile picture.They can be custom photos, uploaded by users, or generated as Identicons as a placeholder.

Image Avatar

Use class avatar. (common) for avatar and classes avatar-lg-size, avatar-md-size, avatar-sm-size or avatar-xs-size for using avatar of different sizes.

Avatar with initials

Use class avatar-initials for avatar with initials.
