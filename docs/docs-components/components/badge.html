Badge

A badge is a counter shown on icons like cart, notification and avatars. It is also used to show online and offline status of a person.

Profile Badge

Profile Badge is used to show the current status of a user. It can be used with 4 variants : online | busy | away | offline

Number badge

To show cart or notification count, you can use number badge.
